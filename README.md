## ✨ Features
* **Real-time Collaboration:** Create and sync study rooms instantly.
* **Modern UI/UX:** Responsive layouts crafted with React, Vite, and Tailwind CSS.
* **Robust Auth:** Secure user registration, session tokens, and login states.
* **Full-Stack Architecture:** Monorepo structure containing both client-side interfaces and backend infrastructure.
